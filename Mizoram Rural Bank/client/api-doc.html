Sarfaesi API Documentation

This document provides a comprehensive reference for all Sarfaesi-related API endpoints, including request and response schemas.

1. Securities Operations

POST /sarfaesi/securities/create
ID: createSecurityDetails
Creates new security details for a Sarfaesi account.
Body: SarfaesiSecurityDetails
PUT /sarfaesi/securities/update/{id}
ID: updateSecurityDetails
Updates existing security details.
GET /sarfaesi/securities/all
ID: getAllSecurities
Retrieves all security details.
GET /sarfaesi/securities/account/{accountId}
ID: getSecuritiesByAccountId
Get security details associated with a specific account ID.
DELETE /sarfaesi/securities/delete/{id}
ID: deleteSecurityDetails
Deletes a security record.

2. Borrower Operations

POST /sarfaesi/borrowers/create
ID: createBorrowerDetails
Creates new borrower/guarantor details.
Body: SarfaesiBorrowerGuarantorDetails
PUT /sarfaesi/borrowers/update/{id}
ID: updateBorrowerDetails
Updates existing borrower details.
GET /sarfaesi/borrowers/account/{accountId}
ID: getBorrowerByAccountId
Retrieves borrower details for a specific account.

3. Approval Operations

POST /sarfaesi/approvals/create
ID: createSarfaesiApproval
Creates a new Sarfaesi approval record.
Body: SarfaesiApproval
GET /sarfaesi/approvals/{id}
ID: getSarfaesiApprovalById
Retrieves specific approval record.
GET /sarfaesi/approvals/region/{region}
ID: getSarfaesiApprovalsByRegion
Lists approvals by region.

4. Account Operations

POST /sarfaesi/accounts/create
ID: createAccount
Creates a new account identification record.
Body: SarfaesiAccountIdentification
POST /sarfaesi/EligibleAccount
ID: sarfaesiEligibleAccount
Marks an account as eligible for Sarfaesi.
PATCH /sarfaesi/update-status/{acctNo}
ID: updateStatusToMovedProceeding
Updates account status to 'Moved Proceeding'.
GET /sarfaesi/eligible-accounts
ID: getEligibleAccounts
Paginated list of eligible accounts.
Query Params: branchId, page, size

5. Workflow Operations

POST /api/sarfaesi/{accountNumber}/initiate
ID: initiateSarfaesiProcess
Starts the Sarfaesi workflow for an account.
POST /api/sarfaesi/{accountNumber}/complete-task
ID: completeCurrentTask
Completes the current workflow task.
Body: TaskCompletionRequest
POST /api/sarfaesi/{accountNumber}/upload-document
ID: uploadDocument
Uploads a document to the case.
GET /api/sarfaesi/{accountNumber}/history
ID: getCaseHistory
Returns full case history including timelines and documents.

6. User Operations

GET /api/sarfaesi/user/tasks
ID: getUserTasks
Retrieves assigned tasks for the logged-in user.
GET /api/sarfaesi/user/dashboard
ID: getUserDashboard
Retrieves aggregated dashboard statistics.
